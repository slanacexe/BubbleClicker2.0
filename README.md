BubbleClicker 🎯

A small browser reaction game inspired by Osu!.

Click the moving bubble as fast as you can before the timer runs out.
Choose your difficulty, enter your nickname, and try to climb up the leaderboard.

🚀 Features

Easy / Medium / Hard difficulty

20-second time limit

Live score & countdown timer

Local leaderboard (saved with localStorage)

Modern animated UI

Built with pure HTML, CSS and Vanilla JavaScript

🕹 How to Play

Enter your nickname

Click PLAY

Select difficulty

Click the bubble as many times as required before time runs out

If you win, your score is saved locally and shown on the leaderboard.

🛠 Tech Stack

HTML5

CSS3 (animated gradient background + glass UI)

Vanilla JavaScript

Browser localStorage
